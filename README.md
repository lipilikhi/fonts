# fonts
লিপিলিখিতে ব্যবহার করার জন্যে বিশেষ ফন্টস।

সকল ফন্ট গুগল ফন্টস (Google Fonts) এবং ওমাক্রনল্যাব (OmicronLab) থেকে সংগৃহীত।
